# EGOP Monorepo

Struktur proyek sekarang dipisah menjadi:

- `frontend/` untuk aplikasi React + Vite yang sudah ada
- `backend/` untuk API/backend yang akan kamu buat nanti

## Menjalankan Frontend

```bash
cd frontend
npm install
npm run dev
```

## Build Frontend

```bash
cd frontend
npm run build
```

## Backend

Folder `backend/` masih kosong dan disiapkan sebagai tempat API, controller, route, dan konfigurasi backend.
